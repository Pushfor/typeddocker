TypedDocker

Watch Star Fork Issue

TypedDocker is a boilerplate we use to build complex web applications composed using various tools and frameworks. We decided that we don't want to be bound to one hosting platform or strategy, one frontend framework or to complex build systems like WebPack which required more maintenance than gave support. Therefore we made very skinny and basic core that uses:

  • Docker
  • TypeScript
  • Gulp
  • Angular 4 - go to example JIT and example AOT
  • React- go to example
  • Sass
  • Golden Layout - example
  • Browserify
  • Node/Express.js
